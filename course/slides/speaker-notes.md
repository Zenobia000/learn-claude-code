# 講稿要點（28 頁 / 60 分鐘）

配速基準：每頁約 2 分鐘。停頓頁 30–60 秒；兩段 demo 各約 4 分鐘。
（p11、p23 demo 前先開好終端機與 repo，`.env` 設定完成。）

## 第一幕：觀念（p01–p07，約 12 分鐘）

**p01 封面**
開場一句：「今天一小時，帶你們從『打 API 的人』變成『設計 agent loop 的人』。」

**p02 你們熟悉的世界**
喚起舊經驗：你們都打過 API、下過 prompt、做過 fine-tuning。
點破：這些都是「單輪」-- 模型回答完，事情就結束了。它沒有手。

**p03 停頓**
唸出問題後安靜 30 秒。收幾個答案再翻頁。
引導語：「它缺的不是智力 -- GPT/Claude 明明看得懂你的程式碼。」

**p04 模型 + Harness**
關鍵句：「Agency 是訓練出來的，這句話今天不展開，記住結論就好。」
主比喻定調：模型是駕駛者，harness 是載具。這門課教造載具。

**p05 五要素**
五個字唸過去即可，各給半句例子。告訴學員：接下來 12 個機制，
每一個都落在這五格的某一格。

**p06 反例**
直接講：「node graph、prompt chain 不是 agent -- 分支是人先想好的。」
對比：agent 的控制流是模型當下決定的。

**p07 停頓**
「最小可行的 agent 要幾行？」讓學員喊數字，通常會猜幾百幾千行。

## 第二幕：作法（p08–p23，約 38 分鐘）

**p08 轉場**
揭曉：30 行。停一拍讓落差發酵。

**p09 Loop 全圖**
指著圖走一遍資料流。強調唯一的退出條件：`stop_reason != "tool_use"`。
關鍵句：「是模型決定何時停，不是你的 if-else。」

**p10 30 行程式碼**
逐段：messages 累積 → 呼叫 → 判斷 → 執行工具塞回去。
結尾預告：「後面 11 個機制，全部疊在這上面，這個迴圈不會再改。」

**p11 Demo 1（約 4 分鐘）**
```
python agents/s01_agent_loop.py
> Create a file called hello.py that prints "Hello, World!"
```
旁白重點：模型自己決定呼叫 bash、自己看結果、自己決定停。

**p12 機制地圖**
「今天不逐一講 12 個，我給你們地圖，深度在課後影片。」三束念一遍。

**p13 骨架層**
s02：加工具 = 加 handler，loop 不動。s03：先計劃再動手，防漂移。
點名：「這就是你們在 Claude Code 看到的 TodoWrite。」

**p14 停頓**
提示語：「messages 只進不出，你們覺得會發生什麼事？」

**p15 Context 工程**
三招一口氣：拆給子代理（乾淨 context）、知識按需載入、滿了就壓縮。

**p16 對應表**
逐行對：Task tool / Skills / /compact。
關鍵句：「教材裡 200 行內的機制，就是產品裡同名功能的原理。」

**p17 任務系統**
檔案式任務圖：可排序、有依賴、重啟不忘。背景執行：慢操作不阻塞。

**p18 停頓**
「一個 agent 不夠用呢？」引導收斂到三個問題：怎麼委派、怎麼溝通、怎麼不打架。

**p19 Multi-Agent I 委派**
每個隊友是獨立 loop、獨立 context；訊息用非同步信箱，不等待。

**p20 Multi-Agent II 協議與自主**
沒有共同協議 = 各說各話。s11 反轉：不派工，隊友自己掃看板認領。

**p21 Multi-Agent III 隔離**
同檔案並行 = 打架。git worktree 一人一份目錄，merge 收斂。

**p22 s_full.py**
「740 行 = Claude Code 雛形。核心迴圈還是那 30 行，其餘全是 harness。」

**p23 Demo 2（約 4 分鐘）**
```
python agents/s_full.py
> （出一個需要拆解、委派的多步驟任務）
```
旁白重點：計劃 → 拆任務 → 委派 → 回收結果。

## 第三幕：目的（p24–p28，約 10 分鐘）

**p24 關鍵洞察**
慢慢唸：「12 個機制疊完，agent loop 一行都沒改。」
「智慧在模型裡，工程在 harness 裡。」

**p25 換模型不換 Harness**
展示 `.env`：MODEL_ID 改一行就換供應商。
「載具是你的資產，駕駛者可以換。」

**p26 課後實作**
三梯度作業 + 14 集影片 + 最小路徑 s01 → s04 → s07。

**p27 總結**
口號收尾：「Agency 是學出來的，Harness 是工程出來的。」

**p28 Q&A**
開放提問；智財聲明頁面停留即可，不需逐字唸。
